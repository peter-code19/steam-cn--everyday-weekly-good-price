# 🎮 Steam 中国区优惠精选

自动获取 Steam 商店打折游戏，按热度/评分/超值排出 TOP 30。

## 快速开始

```bash
python steam-deals.py
```

或双击 `启动.bat`

> 需要 Python 3

## 功能

- 🇨🇳 Steam 中国区真实 CNY 价格
- 🔥按热度 / ⭐按评分 / 💰按超值 三种排序
- ⚡今日特惠 + 🏆本月最值 TOP 30
- 🎁限时免费游戏特殊标记
- 关闭浏览器自动退出，不留后台

## 隐私

- 仅监听 127.0.0.1，外部无法访问
- 数据由浏览器直连 CheapShark / Steam
- 不收集任何用户数据，不需要 Steam 登录
- 源码公开可审查
